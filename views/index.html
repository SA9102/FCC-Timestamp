Timestamp Microservice

A simple program that returns JSON containing the Unix and UTC times of an entered date in the route /api/:date.

Example Usage:

  • [project url]/api/2015-12-25
  • [project url]/api/1451001600000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}

This project is created for freeCodeCamp.org, as part of the Back End Development and APIs challenges.
